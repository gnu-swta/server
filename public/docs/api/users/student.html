학생 정보 요청 특정 학생 유저의 정보를 받아옵니다. /users/student

학생 정보 요청

특정 학생 유저의 정보를 받아옵니다.


GET
http://203.255.43.246:3000/users/student/{student_number}

+ Essential header

Key 설명
Authorization 전달자 "Bearer"을 포함한 JWT을 넘겨줍니다.

+ Query parameters

Parameter 설명
student_number 해당 학번의 정보를 받아옵니다.

+ Response Elements

Value Data type 설명
pk_student INTEGER 학번
number INTEGER 학번 (equal to 'pk_student')
name STRING 이름
department STRING 학과

+ Response Sample

JSON

+ Error code

Code Message 설명
200 - 성공
500 Invaild params 잘못된 parameters 입니다.
500 can't send sql query 내부 DB서버 오류입니다.
500 로그인할 수 없습니다. 기타 이유로 인해 로그인에 실패하였습니다.
